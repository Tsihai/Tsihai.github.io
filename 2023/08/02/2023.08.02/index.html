Java 远程开发

发表于2023-08-02
|字数总计:735|阅读时长:2分钟|阅读量:

使用 IDEA 自带的远程开发功能，可以直接将本地的编译、构建、调试、运行等工作全部都放在远程服务器上执行！

而本地仅运行客户端软件连接服务器，像之前一样编写代码、进行其他开发操作即可。

  • 适合本地电脑性能差（服务器性能强）、开发环境多人协作开发、需要统一开发环境的场景。
